💡 Semantic HTML

Semantic HTML uses tags that clearly describe their meaning. Instead of generic div tags, use <header>, <nav>, <main>, <article>, <section>, and <footer>.

Semantic markup improves accessibility, SEO, and code readability. It helps screen readers understand page structure and assists search engines in indexing content.
