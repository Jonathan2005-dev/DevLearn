Lesson 5: Images and Media

Image tags, alt text, and media elements

📚 Learning Objectives

  • ✓ Embed images with the <img> tag
  • ✓ Understand and use alt text
  • ✓ Control image dimensions
  • ✓ Embed audio and video
  • ✓ Make media responsive

💡 Images and Media

The <img> tag is used to embed images. The src attribute specifies the image file, and alt provides alternative text for accessibility.

HTML5 supports <audio> and <video> tags for embedding media directly without plugins.
