💡 Document Structure

Every HTML document follows a specific structure. The <html> element is the root, containing two main sections: <head> and <body>.

The HEAD section contains metadata and configuration information like the page title, character encoding, viewport settings, and links to stylesheets. This content is not directly visible on the webpage.

The BODY section contains all the visible content of the webpage including headings, paragraphs, images, links, and more.

🔑 Important Meta Tags

<meta charset="UTF-8">

Specifies the character encoding for the document

<meta name="viewport">

Controls layout and scaling on mobile devices

<title>

Sets the browser tab title and page name
