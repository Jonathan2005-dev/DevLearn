💡 Web Accessibility

Accessibility ensures websites are usable by everyone, including people with disabilities. ARIA (Accessible Rich Internet Applications) attributes enhance accessibility for screen readers.

Key practices: provide alt text for images, use semantic HTML, label form fields, ensure keyboard navigation, and use appropriate heading structure.

🔑 Common ARIA Attributes

aria-label

Provides accessible name for elements

aria-required

Indicates form field is required

aria-describedby

Associates element with description
