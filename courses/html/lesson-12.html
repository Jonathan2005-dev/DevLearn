Lesson 12: Final Project

Build a complete HTML website from scratch

📚 Project Objectives

  • ✓ Apply all HTML concepts learned
  • ✓ Create a complete website
  • ✓ Use semantic HTML structure
  • ✓ Implement forms and navigation
  • ✓ Follow best practices and accessibility

📋 Project Requirements

✓ Structure

  • • Use semantic HTML5 tags
  • • Include header, nav, main, footer
  • • Proper heading hierarchy

✓ Content

  • • At least 4 pages/sections
  • • Navigation menu
  • • Contact form

✓ Features

  • • Images with alt text
  • • Lists (ordered/unordered)
  • • Tables or forms

✓ Quality

  • • Proper indentation
  • • Accessibility attributes
  • • Valid HTML5
