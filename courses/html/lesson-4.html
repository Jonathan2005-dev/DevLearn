💡 The Anchor Tag

The <a> tag (anchor) is used to create hyperlinks. The href attribute specifies the destination URL.

You can link to external websites (absolute URLs), other pages on your site (relative URLs), or sections within the same page (anchors).

The target="_blank" attribute opens links in a new tab, while target="_self" (default) opens in the current tab.
