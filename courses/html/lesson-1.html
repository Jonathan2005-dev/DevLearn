💡 What is HTML?

HTML stands for HyperText Markup Language. It's the standard markup language used to create web pages. HTML provides the structure and content of websites, while CSS handles styling and JavaScript adds interactivity.

HTML works by using a system of tags and elements that tell web browsers how to display content. Each tag has a specific purpose, from basic text formatting to embedding media and creating forms.

Think of HTML as the skeleton of a webpage, CSS as the styling (skin, clothes), and JavaScript as the actions (moving, talking, etc.).

🔑 Key Concepts

Tags

HTML elements are defined by tags, which are keywords surrounded by angle brackets: <tag></tag>

Elements

An element consists of an opening tag, content, and a closing tag. Some tags are self-closing.

DOCTYPE

The <!DOCTYPE html> declaration tells the browser this is an HTML5 document.

Structure

Every HTML document has head (metadata) and body (visible content) sections.
